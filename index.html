Convert Google maps saved places to GeoJSON

Works, if places are saved with URL like http://maps.google.com/?q=xx.xxxxxxxxxx,yy.yyyyyyyyyy. Should (but currently does not) filter out other format maps links, thus produced GeoJSON might not be valid (those entries have coordinates array filled with null's)
